Bienvenue sur notre application d'analyse de sentiments

Dans le cadre de notre stage au sein de la Direction Générale des Finances Publiques dans le pôle DTNum, nous avons conçu une application web supportant notre modèle de NLP. Le NLP est une discipline plutôt récente du machine learning permettant de traiter des données textuelles. Ainsi, nous avions pour mission d'implémenter un algorithme de machine learning capable d'interpréter le sentiment dégagée par une phrase : ce sentiment peut être soit positif, soit négatif. Nous vous invitions à explorer davantage cette application pour mieux comprendre ces notions.

L'objectif de cette application est de vous permettre de tester par vous même cette application en choisissant votre modèle de Word Embedding. Vous pouvez également lire notre méthodologie ou nous contacter en cliquant sur les icons correspondant dans la barre de navigation.
